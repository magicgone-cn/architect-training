# 架构师需要什么能力

1. 编写架构文档
2. 设计并实现核心组件或服务
3. 分析性能瓶颈并解决
4. 推动多人多部门的合作
5. 微服务、消息队列、分布式事务
6. 大数据、人工智能、区块链等新技术需要了解

# 面试套路

1. 自我介绍
2. 重点介绍某一个项目经验，从项目背景谈起，面对的问题，解决方法，最好能将总体架构和关键流程画出来。可能会给出一些建议，为什么这样做而不那样做
3. 知识广度的一些问题
4. 手写代码

# 系统架构是什么

架构元素+元素关系（静态+动态）

系统是由什么组成的，他们之间的关系是怎样的

面向多种角色，从不同的视角对其关注点进行描述
